英検3級 文法問題

Grammar Fill-in-the-blank

CEFR: A2 Grade 3

デモ版: 固定の問題1問を体験できます。実際のシステムでは、AIが学生のレベルに応じて動的に問題を生成します。

問題

My sister ( ) to the library every Saturday.

選択肢を選んでください

技術解説

問題生成フロー

1. トピック選択

61種類のトピックエリアから選択

2. 語彙選定

CEFR-J 10,000語データベースから適切な語彙を選択

3. 文章生成

GPT-4oで自然な英文を生成

4. 著作権チェック

過去問との類似度チェック（閾値: 0.85）

5. 選択肢生成

正解1つ + 魅力的な誤答3つ

データベース構造 

// 主要テーブル
eiken_generated_questions
  ├─ id, student_id
  ├─ grade, format
  ├─ question_data (JSON)
  └─ created_at

eiken_vocabulary_lexicon
  ├─ word, cefr_level
  ├─ pos, frequency
  ├─ japanese_meaning
  └─ example_sentence

eiken_topic_areas (61 topics)
  ├─ topic_code
  ├─ topic_name_ja
  ├─ cefr_range
  └─ grade_range

API エンドポイント 

// 問題生成
POST /api/eiken/questions/generate
{
  "student_id": "demo-user",
  "grade": "3",
  "format": "grammar_fill",
  "count": 1
}

// 解答評価
POST /api/eiken/questions/:id/answer
{
  "answer": "B"
}
