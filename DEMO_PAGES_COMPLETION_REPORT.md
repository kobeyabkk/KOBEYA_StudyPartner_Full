### 2. 英検デモページ (`/demo/eiken.html`)

#### 実装された流れ

```
1. 問題表示
   ↓
2. 選択肢をクリック（4択）
   ↓
3. 「解答する」ボタンをクリック
   ↓
4. 正誤判定 + 選択した解答の解説
   ↓
5. 全選択肢の解説表示
   ↓
6. 文法ポイントの説明
   ↓
7. 「もう一度試す」で最初から
```

#### デモデータ
```typescript
問題: "My sister ( ) to the library every Saturday."

選択肢:
A. go       → 三人称単数で不適切
B. goes     → 正解！三人称単数現在形
C. going    → 進行形で不適切
D. went     → 過去形で不適切

CEFR: A2
Grade: 3
Topic: daily-life
文法: 三人称単数現在形
```

#### 技術解説パネル（右サイド）
1. **問題生成フロー**（5ステップ）
   - トピック選択
   - 語彙選定（CEFR-J 10,000語）
   - 文章生成（GPT-4o）
   - 著作権チェック（閾値0.85）
   - 選択肢生成

2. **データベース構造**
   ```
   eiken_generated_questions
   eiken_vocabulary_lexicon
   eiken_topic_areas (61 topics)
   ```

3. **API エンドポイント**
   ```
   POST /api/eiken/questions/generate
   POST /api/eiken/questions/:id/answer
   ```

4. **使用技術**
   - OpenAI GPT-4o
   - Cloudflare D1
   - Hono.js
   - TypeScript

5. **パフォーマンス指標**
   - 問題生成: 2-3秒
   - 著作権閾値: 0.85
   - 語彙データ: 10,000語
   - トピック数: 61

#### UI特徴
- 選択肢ホバーエフェクト
- 正解は緑、不正解は赤にハイライト
- スムーズなアニメーション
- Sticky技術解説パネル

### 3. 小論文デモページ (`/demo/essay.html`)

#### 実装された7ステップ

| ステップ | タイトル | 内容 | フィードバック内容 |
|---------|---------|------|------------------|
| **Step 1** | 読解確認 | お題の理解度チェック | 重要ポイント4つを明示 |
| **Step 2** | 構成計画 | アウトライン作成 | 導入/本文1/本文2/結論の構成提案 |
| **Step 3** | 語彙確認 | 使いたい語彙リスト | 季節/感情/接続詞の語彙リスト |
| **Step 4** | 導入・理由1 | 150字程度の執筆 | 良い点とアドバイス |
| **Step 5** | 理由2 | 100字程度の執筆 | 説得力の評価と改善案 |
| **Step 6** | 推敲 | 全体の見直し | チェックポイント5項目 |
| **Step 7** | 完成 | 最終確認 | 総合評価（構成/表現/語彙） |

#### デモお題
```
タイトル: あなたの好きな季節について

内容:
四季の中で、あなたが最も好きな季節はどれですか？
その季節が好きな理由を、具体的な例を挙げて説明してください。
（300〜400字程度）
```

#### プログレスバー機能
- 7つのステップドット
- 完了: 緑色 + チェックマーク
- アクティブ: 青色 + 光るエフェクト
- 未完了: グレー
- ステップ間の接続線（完了したら緑）

#### ナビゲーション
